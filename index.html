2

Connect to Your VPS

Open a terminal and SSH in:

ssh root@YOUR_IP_ADDRESS

Type yes when asked about the fingerprint, then enter your password.

Windows users: Open PowerShell or install PuTTY. The ssh command works in modern PowerShell too.
3

Create a Telegram Bot

  1. Open Telegram and search for @BotFather
  2. Send /newbot
  3. Pick a display name (e.g., "My AI Bot")
  4. Pick a username (must end in bot, e.g., myname_aibot)
  5. Save the token BotFather gives you
The token looks like: 7123456789:AAH... ‚Äî keep it secret!
4

Run the Setup Script

While SSH'd into your VPS, paste this one command:

curl -fsSL https://raw.githubusercontent.com/YOUR_REPO/main/setup.sh | bash

It will ask you for:

  • Your Anthropic API key
  • Your Telegram bot token
  • A name for your bot
  • Your name
That's it! The script installs everything and starts your bot automatically.
5

Talk to Your Bot!

  1. Open Telegram
  2. Search for your bot's username
  3. Send it a message
First message: You'll get a pairing code. Run this on your VPS to approve it:
clawdbot pairing list telegram
clawdbot pairing approve telegram <code>

Useful Commands

clawdbot status
Check if everything is running
clawdbot health
Detailed health check
clawdbot gateway start
Start your bot
clawdbot gateway stop
Stop your bot
clawdbot gateway restart
Restart your bot

Troubleshooting

Bot doesn't respond

Run clawdbot status to check if the gateway is running.
Run clawdbot health to see if auth is working.
Double-check your API key is correct.

Got a "pairing code" message

This is normal on first contact! Approve it:

clawdbot pairing list telegram
clawdbot pairing approve telegram <code>
Need to start over

Reset everything and re-run:

rm -rf ~/.clawdbot ~/clawd
curl -fsSL https://raw.githubusercontent.com/YOUR_REPO/main/setup.sh | bash
